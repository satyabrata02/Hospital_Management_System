E-Health care management system is command-driven application that helps to manage the record of the patient's as well as generate their hospital bills.

Options | Description

1. Add New Patient Record >> A new patient file can be created by using this module.

2. Add Diagnosis Information >> Adding day-to-day evaluation of the patient or the diagnosis report in the existing file.

3. Add Bed Admission Details >> An existing patient Bed Admission file can be created by using this module.

4. History of the Patient >> Check the previous or existing details of particular patient by using their file name.

5. Bill of the patient >> Generates the bill of the patient with displaying the details.

6. Help >> Provide the help documentation.
